 ## 계산기
  #### 기능 설명
   AC버튼 : 초기화 버튼입니다. <br />
   +/-버튼 : 음수/양수 변환 버튼 입니다. <br />
   %버튼 : 백분율로 변환하는 버튼입니다. <br />
   .버튼 : 소수점 버튼입니다.
   
  #### 프로젝트 정보
   제작 기간 : 3주 <br />
   참여 인원 : 1명 <br />
   사용 기술 : html, scss, css, javascript
